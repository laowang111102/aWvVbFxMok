# Python人脸表情识别系统 python531

## 项目概述


## 技术栈

- **GUI框架**：PyQt5
- **图像处理**：OpenCV
- **深度学习框架**：TensorFlow
- **神经网络模型**：DenseNet121
- **人脸检测库**：face-recognition
- **数据处理**：NumPy
- **图像显示**：Pillow

## 功能模块

### 表情识别

- **图片识别**：支持单张或批量图片上传，自动检测人脸并分析表情。
- **视频分析**：支持导入视频文件，逐帧解析表情变化。
- **实时摄像头检测**：调用摄像头进行实时表情捕捉与分析。

### 表情分类

- **精准识别**：能准确识别7种常见表情，包括生气、厌恶、恐惧、开心、中性、悲伤、惊讶。

### 数据可视化

- **情绪概率可视化**：实时显示每种表情的置信度百分比。
- **表情图标显示**：直观展示识别结果，界面友好。

### 数据输出

- **置信度分析**：输出详细置信度数据，支持结果导出。

## 系统角色

- 用户：通过界面与系统交互，上传图片或视频，查看表情识别结果。
- 系统：负责接收用户输入，调用相关算法库处理数据，返回识别结果。
